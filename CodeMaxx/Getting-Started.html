Getting Started

Read this if you are new to programming or need help

What You'll Need:

An IDE:

This is Software that helps you to write code. Some examples are Visual Studio Code, Brackets, Eclipse, etc.
Also note that there are IDES' for specific languages. Examples are Pycharm for Python, Code::Blocks for C/C++, amongst others.

Developer Tools:

These may include interpreters(e.g. The Python Interpreter), compilers(e.g. GNU G++ for C/C++), extra dependencies, etc.
Don't know which ones you need? Don't worry. They are covered in the tutorials.

The Basics:

Programming:

This is the act of writing a computer program. A program is a software application(or a collection of software applications) that is designed to perform a specific task.

Coding:

This is the process of writing computer software code. Code is an instruction for a computer, that is written in a programming language. A programming language is a package of reserved words that are used to create computer programs.

The Difference Between Them

Coding is a part of programming that deals with writing codes that a machine can understand. Programming is a process that creates programs that involve the validation of codes.

Before You Start

You need to know what you want to do with your chosen programming language(s). Note that you can choose to learn more than one. Consider the list below:

  1. Website Development - HTML, CSS, PHP, Python, JavaScript, etc.
  2. Software Development - C, C++, Java, Python, JavaScript, etc.
  3. Game Development - C++, C#, Python, etc.
  4. Mobile Development - Kotlin(Android), Swift(IOS), Flutter, React Native, etc.

What You Need to Learn:

  1. The basics of programming (Variables, Control Flow, Functions, OOP, etc.)
  2. How to use your chosen IDE.
  3. How to run your code.
  4. If you want to venture into game/app development, you may need to learn some extra things(e.g. Modules for development with Python, Engines for game development, and Libraries for App Development)

    5. All of these will be covered in the tutorials.

Congrats!

You've reached the end of this tutorial. now you should have a basic knowledge of how programming works and how to get started. Good luck!
